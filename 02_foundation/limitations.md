# Limitations

1. Some imported tracks may not match entries in Music4All / Music4All-Onion, reducing usable preference evidence.
2. Recommendation behaviour and output quality are constrained by the availability and quality of feature descriptors in the target dataset.
3. The artefact is limited to a single-user recommendation context and does not evaluate multi-user dynamics.
4. The system does not include collaborative filtering or deep-learning recommenders as baselines.
5. Cross-platform ingestion is intentionally bounded to one practical ingestion path in the MVP.
6. Evaluation is constrained to BSc-feasible testing and does not include large-scale longitudinal or controlled user studies.
7. The contribution is engineering/design evidence for transparent and controllable pipeline construction, not a claim of state-of-the-art recommendation accuracy.

## Interpretation Note

These limitations define the intended validity boundary of the thesis results. Conclusions should be interpreted as artefact-specific design evidence within this bounded scope.
