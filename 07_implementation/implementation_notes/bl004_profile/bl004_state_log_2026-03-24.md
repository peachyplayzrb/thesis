## 3. Processing Logic (Current)
1. Load BL-003 seed table.
2. Filter rows by allowed interaction types.
3. For each included seed row:
   - use `preference_weight_sum` as effective weight
   - aggregate semantic labels from `tags` and `genres`
   - aggregate numeric features by weighted mean
   - aggregate `key` using weighted circular mean over 12 semitones
   - write deterministic seed trace row
4. Build full profile payload (`config`, `diagnostics`, `seed_summary`, numeric + semantic profiles).
5. Build compact summary payload for quick inspection.
