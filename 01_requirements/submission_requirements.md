# Submission Requirements

## Submission Channels
- Required submissions are made via Canvas (formative and summative milestones referenced in handbook/briefs).
- Final project reports are expected to be submitted through Turnitin.
- Project registration, ethics paperwork, milestone submissions, and final submissions all route through Canvas in the module workflow.

## Milestone Pattern (From Handbook)
- Project registration form: submit when a supervisor has agreed to the project.
- Project clearing form: submit only if no project/supervisor has been secured by the deadline.
- Ethical considerations form: submit if applicable.
- Milestone 1: Project proposal report (pass/pass with minor modifications/fail gate).
- Milestone 2: Literature review section (formative).
- Milestone 3: Artefact design and test plan (formative).
- Milestone 4: Professionalism section + early prototype (formative).
- Milestone 5: Draft report + advanced artefact version (formative).
- Milestone 6: Final report + final artefact (summative, component 2).

## Required Final Outputs
- Final project report.
- Final artefact.
- Professionalism companion report (approximately 2000 words) as component 1.
- Project management evidence (for example logbook/progress records/planning evidence).
- Project cover/declaration page using the supplied template.
- Demonstration/viva of artefact before completion (compulsory per handbook language).

## Pass Structure
- Both component 1 and component 2 must be passed to pass the module.
- Component 1 professionalism weighting: 25%.
- Component 2 project including artefact/project management weighting: 75%.
- Milestone misses are formative, but handbook/brief language indicates they can still affect project-management marking.

## Compliance Constraints
- Academic misconduct controls apply (including plagiarism/collusion and inappropriate AI use references in module materials).
- Project declaration/cover page details should follow supplied template.
- The project must remain practical, problem-solving, course-related, and artefact-based with development/testing evidence.

## Unknowns To Confirm
- Exact current academic-year deadlines and milestone dates (Canvas authoritative).
- Any current-year changes to milestone count/structure.
